Online Leave Management System






Language of implementation: PHP, JavaScript, HTML, CSS

Project Submission Details Course Title : Web Database Programming
Instructor : Bijan Paul, Senior Lecturer, CSE Dept
click here for the source codes


Abstract:
Online Leave Management System is developed to keep the informations of employee leaves. Here employees can manage their leaves online and apply for argent leaves. It removes all the difficulties of application for leaves manually. Leave details can be available online. This is an autonomous System.
Introduction:
This website is developed for maintaining database for office employees. The system provides security features through username-password matching where only authorized users can access the system with different authorization level like admin, employee. It maintains workforce efficiency and productivity, optimizes staff scheduling during absences, keeps track of all scheduled employee time off and keep business operations running smoothly.
Scope:
As a whole the system is focused to work with office employees online for any company and on additional modification it can be also used for other purposes.
System Users:
• Admins: Has full privilege on the system's functions
• Employees: Has privilege on some of the system's functions as assigned by the admin

Features:
 Login: The system provides security features through username-password matching where only authorized user can access the system with different authorization level.
 Registered users can -
o check their leaves
o apply for argent leaves
 Super Admin approves admin
 Admin can -
o grant or forbid argent leaves

System Requirement:
 Browser: Chrome, Firefox, Safari, Microsoft Edge.
 RAM: 512mb or higher
 OS: Microsoft windows XP, Vista, 8, 8.1, 10, Linux, Mac-OS
 MySQL server, Xampp, Apache server

Conclusion:
We have tried to make this project as good as possible. It can help any comp to run their leave management system online.
