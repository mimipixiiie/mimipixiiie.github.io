Online Shopping Management System






Language of implementation: PHP, JavaScript, HTML, CSS

Project Submission Details Course Title : Web Database Programming
Instructor : Bijan Paul, Senior Lecturer, CSE Dept
click here for the source codes


Abstract:
Online Shopping Management System is developed to keep the informations and quantity of products. Here usera can order products online. It removes all the difficulties of buying products manually. Price, available quantity, product details can be available online. This is an autonomous System.
Introduction:
Online shopping is a form of electronic commerce which allows consumers to directly buy goods or services from a seller over the Internet using a web browser. Consumers find a product of interest by visiting the website of the retailer directly or by searching among alternative vendors using a shopping search engine, which displays the same product's availability and pricing at different e-retailers.
Requirements Analysis:

FUNCTIONAL REQUIREMENTS:
 Visitors can search and see details of products
 Admin will be approved by super admin
 Admin can update, delete, add products
 Only registered user is allowed to order product
 An admin can also act like a user
 Users can update their profile
 User can cancel products under some constraint
 User can bookmark pages and can see the cart items.

NON FUNCTIONAL REQUIREMENTS:
 The system will be available for 24 hours
 The system is developed with PHP, CSS3,HTML5 and JAVASCRIPT
 User will not be able to cancel products after shipment
 User will be able to order any amount of products as long as product is in stock
 The system will be secured with session management
 Users information will be kept safe

Scope:
As a whole the system is focused to work with users online for any shops and on additional modification it can be also used for other purposes.
System Users:
• Admins: Has full privilege on the system's functions
• Users: Has privilege on some of the system's functions as assigned by the admin

Description:
This website is developed for online shopping. It would make categorized searching, viewing and selection of a product easier for the users. Users can search products based on age, sex or category, add the product into wishlist, add products into cart with any amount, place order & also cancel it on terms etc. Admin can add any product into database, entry the payments etc.
